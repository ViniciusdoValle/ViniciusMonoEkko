# ViniciusMonoEkko
Site do Ekko meu mono no LoL que criei pra treinar e estudar programação.
 Adicionei imagens (splash art, skins, runas, builds), vídeos (combos e um penta que fiz), explicações sobre runas, builds e itens. Também incluí minha história no LoL, motivos de ser mono Ekko, lore, curiosidades, counters, fraco contra ele, como jogar, função, dificuldade, detalhes das skills com vídeos e falas + interações.
